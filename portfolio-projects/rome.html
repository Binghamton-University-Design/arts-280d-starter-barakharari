ROME

ROME

Eye movement tracking software written in Python

Description

ROME is an eye-movement tracking software for neurological health assessment. I programmed 5 block moving tests meant to challenge eye movement in order to detect potential brain abnormalities.

The project was developed with 4 biomedical engineers as part of a senior project.

In addition, eye-movement results are stored locally in order to calculate empirical averages and standard deviations for different age groups
